## Accessibility Considerations

### Keyboard Navigation

```
Tab Order:
1. Filter controls
2. Sort controls
3. MR list items (each focusable)
4. Action buttons within items
5. Pagination controls

Shortcuts:
- Enter: Open selected MR
- Space: Select/deselect MR (for bulk actions)
- â†‘â†“: Navigate list
- Esc: Close dialog/cancel action
```

### Color Contrast

```
All text must meet WCAG AA:
- Normal text: 4.5:1 contrast ratio
- Large text: 3:1 contrast ratio
- Don't rely solely on color to convey information
- Use icons + text for status indicators
```
